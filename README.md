# 📈 KOSPI 다음날 종가 예측 머신러닝 프로젝트  

---

## 1. 프로젝트 개요  
- 시계열 기반 딥러닝 모델인 LSTM와 다양한 금융 지표를 이용하여 **다음 날 KOSPI의 종가를 예측하는 머신러닝 모델**을 만드는 것을 목표로 한다.

- 데이터 전처리 → 모델링 → 모델 학습 → 성능 평가 로 전체 머신러닝 파이프라인을 구성했다.

- 프로젝트 기간: 2025.11.04 - 2025.11.24



## 2. 역할 분배
- 강민석: 프로젝트 리드 및 데이터 모델링
- 김기성: 데이터 전처리 
- 염지인: 데이터 시각화 및 발표자료 준비
- 정수원: GitHub 관리 및 각종 문서화



## 3. 사용 데이터  
- yfinance 라이브러리를 활용
- 2005년 1월부터 2025년 11월 3일까지(약 20년)의 일봉 데이터를 수집함.
- KOSPI 외 거시 경제 변수를 포함한 총 9개의 특성(feature)을 사용함.  


| 변수명 | 설명 |
|--------|------|
| KOSPI_Open | 코스피 시가 |
| KOSPI_Close | 코스피 종가 |
| KOSPI_High | 코스피 고가 |
| KOSPI_Low | 코스피 저가 |
| KOSPI_Volume | 코스피 전체 거래량 |
| SP500 | 미국 대표 주가지수 |
| VIX | 미국 S&P500 옵션 가격 기반 변동성 지표 |
| USD_KRW | 원/달러 환율 |
| US_10Y_Bond_Yield | 미국 10년물 국채 금리 |
| 변동량 | 시가 대비 종가 변동 값 |



## 4. 데이터 전처리  
- 결측치 제거 및 보정  
    + 경제학적 시차 적용: 미국 시장(SP500, VIX 등)의 정보는 한국 시간 기준 다음 날 KOSPI에 영향을 미침.
- Min-Max Scaling 적용  
- 슬라이딩 윈도우 방식으로 5일 묶음 구조 생성: **과거 5일치 입력 -> 다음날 종가 예측**
- 학습 / 검증 / 테스트 세트 분할  



## 5. 모델 아키텍처  
- 입력 데이터: (None, 5, 9) 형태의 텐서(Tensor)
- 아키텍처: TensorFlow/Keras를 사용함.

| Layer        | Shape         |
| ------------ | ------------- |
| Input        | (3472, 5, 9)  |
| ↓            |               |
| LSTM(64)     | (None, 5, 64) |
| ↓            |               |
| Dropout(0.2) | (None, 5, 64) |
| ↓            |               |
| LSTM(32)     | (None, 32)    |
| ↓            |               |
| Dropout(0.2) | (None, 32)    |
| ↓            |               |
| Dense(1)     | (None, 1)     |

## 7. 결론
**수정필요**  
LSTM은 9개 변수 간의 복잡한 비선형 관계 및 경제학적 시차를 '선택적 기억' 메커니즘으로 학습하여, ARIMA 대비 오차율을 82.2% 대폭 개선하며 실제 KOSPI 추세를 성공적으로 추종(RMSE 105.4)할 수 있음을 보였다.그래프에서 확인되듯, 변동성이 컸던 202X년 X월 구간에서 ARIMA와 달리 LSTM은 외부 변수(VIX, S&P)의 움직임을 참고하여 실제 급등락 패턴을 예측하는 데 더 강점을 보였다.



## 8. 향후 개선점
- 하이퍼파라미터 튜닝: WINDOW_SIZE (현재 5일)를 10일, 30일로 변경하거나 LSTM 레이어를 더 깊게 쌓아 추가 성능 확보.
- 문제 정의: '가격 예측'(회귀)이 아닌, '내일 오른다/내린다'(분류) 문제로 전환하여 정확도 기반으로 모델을 재평가.
